## Description
A food recipe website where users can explore and cook a variety of meals.

## Tech Stack
- React
- HTML
- CSS
- Bootstrap
- JavaScript

## Features
- Responsive UI
- User-friendly interface
- Interactive Recipe Cards
- 
- Fast loading and lightweight design

## How to run
1. Install all the project dependencies required to run the application.

`npm install`

2. Runs the app in development mode.

`npm start`

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
